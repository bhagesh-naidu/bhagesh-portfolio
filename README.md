# Bhagesh Portfolio

This repository contains a personal portfolio website built with modern web tooling.

## Overview

- Framework: React with Bun and Vite
- Language: TypeScript
- Purpose: Showcase portfolio work, projects, and contact information

## Development

Install dependencies:

```bash
bun install
```

Run the development server:

```bash
bun run dev
```

Build for production:

```bash
bun run build
```

## Repository Structure

- `src/` - application source code
- `src/routes/` - route components for the site
- `src/components/` - reusable UI components
- `package.json` - project scripts and dependencies
- `bunfig.toml` - Bun configuration
